# Playground – Language Learning Platform

Just a playground for me to learn and test (frontend & backend).  
The website is designed for junior Japanese and English learners.

**Live Demo:** https://aceplayground.dev

## Tech Stack

- Frontend: React, TypeScript, Vite, Material UI
- Backend: ASP.NET Core (.NET 8), C#
- Database: PostgreSQL
- Auth: Firebase Authentication (Email/Password + Google)
- Storage: Supabase Storage (avatars)
- Hosting: Vercel (frontend), Railway (backend)
- Testing / Debugging: Postman, Swagger, Browser DevTools, React DevTools
